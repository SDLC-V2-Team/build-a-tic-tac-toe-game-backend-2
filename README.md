# Tic Tac Toe (Command-Line)

A simple command-line tic-tac-toe game where you play against the computer.

## How to Run

```bash
python main.py
```

Requires Python 3.6+ (no external dependencies).

## How to Play

- You are **X**, the computer is **O**.
- The board positions are numbered 1–9 as shown below:
  ```
   1 | 2 | 3
  ---+---+---
   4 | 5 | 6
  ---+---+---
   7 | 8 | 9
  ```
- Enter a number (1–9) to place your mark in the corresponding cell.
- The game ends when one player wins or the board is full (draw).
